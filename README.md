# About CEnTR*IMPACT

CEnTR*IMPACT was developed to support community-engaged scholars in:

* evaluating projects
* reflecting on partnerships
* communicating impact
* visualizing project dynamics
* supporting narrative reporting

The framework was designed in response to growing institutional and funding demands for quantitative indicators while preserving the complexity, relationality, and contextual nature of community-engaged research.

The ensemble includes:

* Direct Indicators
* Alignment Scores
* Project Dynamics Scores
* Cascade Effects Scores

Importantly, CEnTR*IMPACT was designed as:

* a heuristic system
* a reflective infrastructure
* a storytelling support framework

and NOT as:

* a ranking engine
* a high-stakes evaluation mechanism
* a replacement for qualitative narratives
* an automated decision-making system
